## 📌 Overview

Terraform module for Google Kubernetes Engine:

- Scope: standard private cluster
- Features:
  - **Networking**: VPC-native, private nodes, DNS control-plane endpoint, network policy
  - **Security**: Shielded nodes, secure boot, Workload Identity, workload metadata protection
  - **Autoscaling**: Node pool autoscaling, Horizontal Pod Autoscaling, Vertical Pod Autoscaling
  - **Upgrades**: auto-repair, manual-upgrade
